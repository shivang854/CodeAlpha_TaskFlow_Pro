# TaskFlow Pro – Project Management Tool

TaskFlow Pro is a full-stack project management web application built for **CodeAlpha Full Stack Development Internship – Task 3**.

## 🚀 Features
- User Registration / Login / Logout
- Create and manage projects
- Add tasks inside projects
- Edit task details
- Track task status and progress
- Add comments on tasks
- Modern dark UI with glassmorphism design

## 🛠️ Tech Stack
- **Frontend:** HTML, CSS, JavaScript
- **Backend:** Django (Python)
- **Database:** SQLite3

## 📌 Functionalities
- Secure authentication system
- Personal project dashboard
- Task creation and task detail pages
- Comment system for collaboration
- Responsive modern interface

## 📂 Project Structure
- `taskflow/` → main Django project
- `manager/` → app for projects, tasks, and comments
- `templates/manager/` → HTML templates
- `static/manager/` → CSS styling

## ▶️ How to Run
```bash
python -m venv venv
venv\Scripts\activate
pip install django
python manage.py makemigrations
python manage.py migrate
python manage.py runserver
